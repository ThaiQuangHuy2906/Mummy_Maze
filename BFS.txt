Algorithm: Hard Mummy Movement (BFS)
Input: CurrentPos, PlayerPos, MazeMap
Output: The next cell to move to

1. IF distance(CurrentPos, PlayerPos) == 1 THEN
2.    RETURN PlayerPos (Attack player directly)
3. END IF

4. Initialize Queue Q, push CurrentPos into Q
5. Initialize Visited map, mark CurrentPos as visited
6. Initialize Trace map to reconstruct path
7. TargetCell = NULL

8. WHILE Q is not empty DO
9.    Current = Q.pop()
10.   IF distance(Current, PlayerPos) == 1 THEN
11.       TargetCell = Current
12.       BREAK Loop
13.   END IF
14.   FOR each Direction (Up, Down, Left, Right) DO
15.       NextCell = Current + Direction
16.       IF isValidMove(Current, NextCell) AND NextCell not Visited THEN
17.           Mark NextCell as Visited
18.           Trace[NextCell] = Current
19.           Q.push(NextCell)
20.       END IF
21.   END FOR
22. END WHILE

23. IF TargetCell is found THEN
24.    // Backtrack to find the first step
25.    Step = TargetCell
26.    WHILE Trace[Step] is not CurrentPos DO
27.        Step = Trace[Step]
28.    END WHILE
29.    RETURN Step
30. ELSE
31.    Skip Turn
32. END IF
